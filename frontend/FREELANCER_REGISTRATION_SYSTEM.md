# 自由职业者注册和验证系统

## 概述

这是一个为中国自由职业者平台设计的全面注册和验证系统，包含多步骤引导、专业信息收集、技能评估、作品集管理和身份验证等功能。

## 部署和使用

### 1. 安装依赖
```bash
npm install
# 或
yarn install
```

### 2. 环境配置
- 配置API端点
- 设置文件上传限制
- 配置短信服务

### 3. 启动开发服务器
```bash
npm run dev
# 或
yarn dev
```

### 4. 构建生产版本
```bash
npm run build
# 或
yarn build
```
